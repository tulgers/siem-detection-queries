# Brute Force Investigation

## Scenario
User shows multiple failed login attempts.

## Steps

1. Validate the alert  
Check logs for repeated failed login attempts.

2. Analyze failed logins
3. Check for successful login  
4. Investigate context  
- IP reputation  
- Location  
- Device  
- User behavior  

5. Decision  
Determine if it is a false positive or a real threat.

## Note
Sorting logs by time helps focus on the most recent activity during the investigation.
